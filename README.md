# Deep Focus Tracker

A simple mobile application aimed to help easily track what work you've been focusing on for the day and maintains a calendar for each focus session

This is an [Expo](https://expo.dev) project created with [`create-expo-app`](https://www.npmjs.com/package/create-expo-app).

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
